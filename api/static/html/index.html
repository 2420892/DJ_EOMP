USERS ROUTE

Methods Path Description
GET / Get to our home page
GET /users Retrieve a list of users.
GET /user/id Retrieve a single user.
POST /register Create a new user.
POST /login Login
PATCH /users/:id Update user information.
DELETE /users Deletes all users
DELETE /user/id Deletes a single user

PRODUCTS ROUTE

Methods Path Description
GET /products Get all our products
GET /product/id Retrieve a single product
GET /product/category Get product by specified category(Watches,Rings,Necklaces)
GET /product/gender Get product by specified gender(female,male)
POST /addproduct Create a new product.
PATCH /product/:id Update product information.
DELETE /product/id Deletes a single product
DELETE /products Deletes all products
